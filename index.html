HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An HTML developer is responsible for providing the structure for the webpage and making sure it provides a positive user experience.

GIT

  • GIT is a version control system that lets you manage and keep track of each version of your project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds modifications in the current working branch and adds all changes
  • git commit -m "task of what you're doing": describes what the commit contains
  • git pull origin main: receives a branches modifications into the local environment, indicates source of pull will be in the GitHub repo, Main branch
  • git push origin branch name: pushes changes locally to our remote GitHub branch
